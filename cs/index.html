Small Shell in C

OSU Operating Systems Project

I wrote a simplified bash-like shell in C for my Operating Systems course. It supports the built-in commands exit, cd, and status, redirection of standard input and standard output, comments, some variable expansion, and foreground and background processes, along with a few other bells and whistles. This was a neat way to get more practice with C strings, file I/O, signals, forking processes, and dealing with zombie processes.

File Sharing Network Application

I wrote a file transfer program for my Networks course consisting of a file transfer server written in C and a file transfer client written in Python. The server and client establish two TCP connections: a control connection for sending/receiving commands and error messages, and a data connection for sending/receiving data such as text files. An interesting problem that came up was making sure large files were received perfectly intact, especially when the server sent data faster than the client could receive it.

Instant Messaging Network Application

This is a simple chat system consisting of a chat server written in Python and a chat client written in C. The server program waits on a specified port for client requests. The client connects on the specified port, and the two hosts then alternate sending and receiving messages until one closes the connection. After the connection is closed, the server is available to accept other connections. This project, which I completed for my Networks course, was a fun way to gain familiarity with TCP and the sockets API.

Random Arrays with MASM

For my x86 Assembly Language course, I wrote a small MASM program that takes a user-input positive integer n and generates n random numbers in an array. It then outputs the unsorted array, the median value, and the array sorted in descending order. This was a neat way to practice working with procedures, passing parameters, data validation, arrays, and indirect addressing in assembly.

Matrix Math Shell Script

To become more familiar with Unix utilities and shell programming, I wrote a bash shell script that performs matrix operations using input from a file or stdin. The script can print the dimensions of a matrix, transpose a matrix, calculate the mean vector of a matrix, add two matrices, and multiply two matrices.

CUDA Monte Carlo Simulation

In this project for my Parallel Programming course, I took a scenario involving a laser beam pointed at a circle that is randomly changing size and location, and developed a Monte Carlo simulation for it to determine the probability of the beam ricocheting off the circle and hitting a plate below. Running the hundreds of thousands of trials in this simulation was a good use for GPU parallelism! I used C++ and CUDA to pass data between the CPU and GPU, and run and time the simulation.
